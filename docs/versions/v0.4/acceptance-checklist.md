# KnowNest V0.4 Acceptance Checklist

## 1. 验收范围

本清单用于验证 V0.4：AI 类型模板化整理。

验收依据：

- `docs/KnowNest_V0.4_PRD.md`
- `docs/versions/v0.4/technical-design.md`
- `docs/versions/v0.4/implementation-plan.md`

验收环境至少包括：

- 本地开发环境。
- 生产或预览环境。
- 一个已有历史知识数据的账号。
- 一个首次使用模板功能的新账号。

## 9. 本地验证命令

交付前建议运行：

```bash
npm run lint
npm run test:knowledge-templates
npm run test:ai
npm run test:knowledge-items
npm run test:tags
npm run test:categories
npm run test:knowledge-item-draft
npm run test:knowledge-list-item
npm run build
```

如果 `test:knowledge-templates` 尚未加入 `package.json`，应先按实施计划补齐。

## 10. 完成判定

V0.4 可以判定完成的条件：

1. 模板管理、类型默认模板、AI 按模板整理、前后对比和分区应用全部验收通过。
2. 模板和知识权限隔离通过验证。
3. V0.1、V0.2、V0.3 已有核心功能无回退。
4. 本地最小验证命令通过。
5. 生产或预览环境完成数据库迁移后，真实账号可完整走通 V0.4 主流程。
